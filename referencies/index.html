    Referències del Projecte Intermodular

    2n d'ASIX – Auditoria de Seguretat i Ciberseguretat
    Data: [Data actual]

    1. Documentació Oficial i Eines Utilitzades

    1.1 Metodologies i Gestió de Projectes

    • Scrum Guide – Guia oficial de Scrum: scrumguides.org
    • Kanban – Gestió visual de tasques: kanbanize.com
    • Trello – Eina de gestió de projectes utilitzada: trello.com

    1.2 Desenvolupament i Programació

    • Python 3.9 – Documentació oficial: docs.python.org
    • Docker – Documentació oficial: docs.docker.com
    • Dockerfile Reference – Guia de referència: Dockerfile reference

    1.3 Ciberseguretat i Auditoria

    • theHarvester – Eina OSINT: GitHub – laramies/theHarvester
    • Shodan API – Documentació oficial: developer.shodan.io
    • SSH Audit – Eina d'auditoria SSH: GitHub – jtesta/ssh-audit
    • Nmap – Escaneig de xarxes: nmap.org

    1.4 APIs i Comunicacions

    • Telegram Bot API – Documentació oficial: core.telegram.org/bots/api
    • Python-Telegram-Bot – Biblioteca per a bots de Telegram: python-telegram-bot.org

    1.5 Disseny Web i Documentació

    • Markdown Guide – Sintaxi Markdown: markdownguide.org
    • GitHub Pages – Allotjament de la landing page: pages.github.com
    • HTML i CSS – Referències MDN: developer.mozilla.org

    2. Fonts d’Informació i Formació

    2.1 Cursos i Certificacions de Referència

    • CompTIA Security+ – Certificació bàsica en ciberseguretat: comptia.org
    • CEH (Certified Ethical Hacker) – Formació en pentesting: www.eccouncil.org
    • CISSP – Certificació avançada en seguretat: www.isc2.org

    2.2 Plataformes d’Aprenentatge

    • Coursera – Cursos en ciberseguretat i xarxes: coursera.org
    • Udemy – Cursos pràctics de Docker, Python i seguretat: udemy.com
    • Cybrary – Formació especialitzada en ciberseguretat: cybrary.it

    2.3 Informació de Mercat Laboral

    • InfoJobs – Ofertes de feina del sector TIC a Catalunya: infojobs.net
    • LinkedIn – Tendències del sector i competències demandades: linkedin.com
    • Institut Nacional de Seguretat i Salut Laboral – Informació sobre riscos laborals: insst.es

    3. Legislació i Normatives Rellevants

    • Llei Orgànica de Protecció de Dades (LOPD) – Regulació espanyola de protecció de dades.
    • Reglament General de Protecció de Dades (RGPD) – Regulació europea de privadesa.
    • Llei de Serveis de la Societat de la Informació (LSSI) – Regulació de serveis digitals a Espanya.
    • ISO/IEC 27001 – Estàndard de gestió de seguretat de la informació.

    4. Bibliografia Complementària

    • “Python for Pentesters” – Justin Seitz (2020).
    • “The Web Application Hacker’s Handbook” – Dafydd Stuttard i Marcus Pinto (2011).
    • “Docker Deep Dive” – Nigel Poulton (2020).
    • “Metodologies Àgils: Scrum i Kanban aplicats” – Lluís Casals (2019).

    5. Recursos Oberts i Comunitats

    • GitHub – Repositoris de codi obert relacionats amb seguretat: github.com
    • Stack Overflow – Comunitat de suport en desenvolupament: stackoverflow.com
    • Reddit – r/netsec i r/cybersecurity – Comunitats professionals: reddit.com/r/netsec

    6. Agraïments

    Aquest projecte s’ha beneficiat de les aportacions de:
    - Professorat del cicle d’ASIX de l’Institut de l’Ebre.
    - Comunitat de desenvolupadors de codi obert que manté les eines utilitzades.
    - Col·legues i companys del projecte per les revisions i suggeriments.

    Totes les referències s’han utilitzat amb finalitats educatives i de documentació tècnica.*
